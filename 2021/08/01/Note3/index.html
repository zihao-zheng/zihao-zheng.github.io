Error Analysis

Carrying out error analysis

TP（True Positives）我们倒着来翻译就是“预测为正样本，并且预测对了”（真阳性）
TN（True Negatives）意思是“预测为负样本，而且预测对了”（真阴性）
FP（False Positives）意思是“预测为正样本，但是预测错了”（假阳性）
FN（False Negatives）意思是“预测为负样本，但是预测错了”（假阴性）

Mismatched training and dev/test set

Training and testing on different distributions

当训练集和测试集不匹配时
98%train（web） 1%dev(mobile) 1%test(mobile)

bias and variance with mismatched data distributions

数据分布不匹配的偏差和方差
1.human error 1-2.avoidable variance 估计优化上限
2.Train error 2-3.variance 方差，反映了能将训练集多好地泛化到train-dev set
3.Train-dev error 3-4.data mismatched 衡量数据匹配度
4.dev error（开发集）4-5.degree of overfitting to the dev set 算法对开发集的过拟合，如果太大，说明神经网络太偏向开发集了，可能需要找一个更大的开发集（因为dev和test集同分布，所以唯一可能的原因是对开发集的过拟合）
5.Test error

Adressing data mismatch 解决数据不匹配的问题
