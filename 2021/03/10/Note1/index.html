向量化实现反向传播

最终的向量化效果

original:
$$
\begin{array}{l}
J=0, \quad d w_{1}=0, \quad d w_{2}=0, \quad d b=0 \
\text { for } i=1 \text { to } m: \
\qquad \begin{array}{l}
z^{(i)}=w^{T} x^{(i)}+b \
a^{(i)}=\sigma\left(z^{(i)}\right) \
J+=-\left[y^{(i)} \log a^{(i)}+\left(1-y^{(i)}\right) \log \left(1-a^{(i)}\right)\right] \
d z^{(i)}=a^{(i)}-y^{(i)} \
d w_{1}+=x_{1}^{(i)} d z^{(i)} \
d w_{2}+=x_{2}^{(i)} d z^{(i)} \
d b+=d z^{(i)}
\end{array} \
J=J / m, \quad d w_{1}=d w_{1} / m, \quad d w_{2}=d w_{2} / m \
d b=d b / m
\end{array}
$$

Last

$$
\begin{array}{l}
z=w^{\top} X+b \
=n p \cdot d o t(w \cdot T, x)+b \
A=\sigma(z) \
d z=A-Y \
d w=\frac{1}{m} X d z^{\top} \
d b=\frac{1}{m} n p \cdot \operatorname{sum}(d z) \
w:=w-\partial d w \
b:=b-\partial d b
\end{array}
$$
